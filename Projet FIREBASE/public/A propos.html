Qui sommes-nous ?

Nous sommes un groupe de quatre étudiants en 2nde année de D.U.T. Mesures Physiques à l'I.U.T. de Créteil-Vitry à l'UPEC.
Les objectifs de ce site web sont répertoriés ci-dessous :

Base de données

Le premier objectif est de créer une base de données à l'aide de Firebase afin de recevoir les données issues des différents capteurs en temps réel et horodatées.

Récupération des données

Le second est de récupérer les données issues de la base de données Firebase et de les stocker dans un tableau.

Traitement des données

Enfin, le site web a pour but de traiter les données de manière intelligente. Nous avons pour cela mis en place, par exemple, l'envoi d'un mail automatique à l'opérateur lorsque l'on obtient une valeur de Courant supérieure à X Ampères.
